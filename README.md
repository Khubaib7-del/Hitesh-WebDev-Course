## 🚀 About The Repository
This repository contains the complete codebase and project history from **Hitesh Choudhary's Complete Web Development Course** on Udemy. It serves as a structured timeline of my learning, starting from basic HTML/CSS all the way to advanced backend databases and full-stack projects.

To practice real-world software engineering workflows, each section of this course is developed on its own dedicated branch and merged via **Pull Requests (PRs)**.

## 🛠️ How to Run Locally

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your local machine.

### Instructions
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Khubaib7-del/Hitesh-WebDev-Course.git
   cd Hitesh-WebDev-Course
   ```

2. **Explore a Section Branch:**
   To view a specific section's branch before it was merged:
   ```bash
   git checkout branch-name
   ```
   *(For example: `git checkout 01-html-css`)*

3. **Running the Node.js Projects:**
   Navigate into a project directory (e.g., `05-NODE_JS/todo`):
   ```bash
   cd 05-NODE_JS/todo
   node todo.js
   ```
